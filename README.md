# Curtales Home Sales Analytics

Sales data pipeline for generating, cleaning, and analyzing sales data with Power BI dashboard.

## Files

- `data_generator.py` - Generate sample sales data
- `data_cleaner.py` - Clean and transform data for Power BI

## Usage

1. Run data_generator.py to create sample data
2. Run data_cleaner.py to clean and prepare data
3. Load cleaned CSV files into Power BI
